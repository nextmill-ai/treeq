# Spartan Tree & Landscape — Pricing Reference (Extracted)

Source: `Spartan_Pricing_Reference.xlsx` (11 sheets), extracted 2026-05-12. Verbatim where useful.

## 1. Labor rates (hourly, per worker)

- **Ground Worker** — $125/hr, 1-hr min. Per worker; each additional ground worker bills independently.
- **Climber (ANSI Z133 certified)** — $250/hr, 1-hr min.
- **Bucket Truck w/ Operator** — $250/hr, **2-hr min** (operator included).
- **Chip Truck w/ Operator** — $250/hr, 1-hr min (chipper included).
- **Equipment w/ Operator** (skid steer, mini-ex, telehandler) — $250/hr, 1-hr min.
- **45-Ton Crane w/ Operator** — $500/hr, **4-hr min** (setup/breakdown billable).
- **Yard Cleanup** — $95/hr, 2-hr min.

No apprentice tier. Line-item minimums do NOT cascade.

## 2. Crew composition pattern

Workbook does NOT define named crew codes (no "3 ALPHA 2" style). Quote is built bottom-up: Bucket Truck + Climber + N Ground Workers + Chip Truck, each on its own hourly line.

## 3. Service-line pricing

### Tree Removal
Quote = **MAX(hourly buildup, highest applicable floor)**, then hazard modifier, then job multipliers.

Floors (highest applies):
- Height — no rigging $10/ft / moderate $20/ft / complex $30/ft
- DBH — no rigging $50/in (>16" DBH) / moderate $100/in / complex $125–$150/in (worksheet uses $137.50)
- Codominant leads: 2.0x for 2 leads, up to 3.0x for 3+
- Loader required: $750 flat min
- One log load: $1,000 min. Multiple loads: $2,000/load min.
- Log truck disposal: $500/60-yd load (when >2 loads)

### Takedowns (cut & leave)
$100 bottom no rope / $150 wedge / $200 pull rope / $300 through pulley in woods / **$400 floor any tree >24" DBH** (use higher of method vs DBH floor).

### Tree Trimming
Cut-count floors: 1 cut $100 / 2 close $150 / cut req rigging $200 / single cut high lifeline $200 / multi-cut high lifeline $300. Working height: ≥30 ft = $300 / ≥40 ft = $400.

**Limb-as-Tree rule:** any lead with ≥8" union diameter priced as its own tree ($10/ft union-to-tip plus DBH floor if ≥16"), ADDED on top of trim work (stacks across multiple leads).

### Stump Grinding
- Primary (largest): MAX($100, $100 + $11 × (Dia − 12)) — $100 covers first 12", then $11/inch.
- Add-on (each other): MAX($25, $5 × Dia).
- Reference ladder: 6"=$100/$30, 12"=$100/$60, 14"=$122/$70, 18"=$166/$90, 24"=$232/$120, 30"=$298/$150, 36"=$364/$180, 48"=$496/$240, 60"=$628/$300.
- Large Stump >18" requiring >1hr: $300/hr alternate row.
- **Grinding Removal (haul grindings):** = total grind price; **2x if no truck access**; $0 if grindings left.
- Always measure root-flare diameter at ground level.

### Planting (markup on plant cost)
- <$1,000 plant cost: **3x**
- $1,000–$2,000: **2.5x**
- >$2,000: **2x**

### Hardscape / Landscape
- Mulching $160/yd, 3-yd min.
- Edging & Prep $90/yd, 3-yd min.
- Stone Install $160/yd, 3-yd min.
- Polymeric Sand $6/sqft, 30-sqft min.
- Powerwashing $1/sqft, 200-sqft min.
- Topsoil delivered (no install) $450/yd, 3-yd min.
- Topsoil + Seeding $500/yd, 3-yd min.

### Land clearing (per sqft, 250-sqft min)
Light ground cover $1 / dense ground cover $2 / light brush $3 / dense brush $4. **Forestry Mulching** $500/hr, 8-hr (full day) min.

### PHC (Plant Health Care)
- EHS Static Cabling $300/unit, 2-unit min.
- COBRA Dynamic System $400/unit, 2-unit min.

### Snow removal
**Not in workbook.** No line items defined.

## 4. Markup / margin targets
No explicit margin % stated for tree work. Planting carries 2–3x markup. Bag topsoil reference: retail $3–5/40-lb bag; bulk ~$30–50/cu yd Rochester.

## 5. Material details
Topsoil per stump: hole dia = stump dia + 12" (root-flare margin); cap depth 3"; full backfill 6"; 0.75 cu ft/40-lb bag; 27 cu ft/yd. Example: 24" stump = 36" hole → 3 bags cap / 5 bags full; 48" stump → 7 bags cap / 14 bags full.

## 6. Disposal / haul
- Chip/brush — bundled in Chip Truck hourly.
- Logs — $250/load haul-away. Floors: 1 load $1,000 min; 2+ loads $2,000/load; log truck (60-yd) $500/load when >2 loads.
- Grindings — equal to grind price; **doubled if no truck access**.

## 7. Add-ons / modifiers

**Hazard modifiers (work portion only — labor + equipment hours + tree-removal/limb-as-tree floors; NEVER stack, take highest):**
- Over Obstacle (house, garage, shed, fence, pool, driveway, vehicle, hardscape): **2.0x**
- Over House Power (service drop ~240V): **2.0x**
- Over High Power (primary distribution 7.2kV+): **3.0x** — requires LCQ arborist per ANSI Z133.

**Job multipliers (on subtotal, after hazard):**
- Poison Ivy +50% (×1.5)
- Leave-All: −25% to −75% (×0.75 to ×0.25), judgment-based on what's left
- Leave Half Load logs: −$150 each
- Leave Full Load logs: −$300 each
- No Small Cleanup (<1" debris): −10% (×0.90)
- **Priority Scheduling: +25%** applied LAST to post-multiplier total

**Other:**
- No-truck-access on Grinding Removal: ×2 (independent, can coexist with hazard)
- Stacking order: Hazard → subtotal → percentage multipliers (chain) → dollar discounts → Priority Scheduling
- Hazard does NOT apply to: stump grinding, log haul, topsoil, materials, trip charges

## 8. Log volume calc

For load-count floors: solid cu yd = π × (DBH/24)² × log_ht_ft / 27; ×1.3 stacked. Dump trailer = 14 cu yd; log truck = 60 cu yd.
